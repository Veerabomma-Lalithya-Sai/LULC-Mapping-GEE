# LULC Mapping using Machine Learning

## Code
The implementation was developed using Google Earth Engine (JavaScript API).  
The code is provided in `lulc_mapping.js`.

## Overview
This project performs Land Use and Land Cover (LULC) classification using Sentinel-2 satellite imagery and Random Forest algorithm in Google Earth Engine.

## Features
- Cloud masking and preprocessing of satellite imagery
- Feature selection using spectral bands
- Supervised classification using Random Forest
- Accuracy evaluation using confusion matrix

## Results
- Achieved 84% classification accuracy
- Successfully classified land into water, vegetation, built-up, and barren areas

## Tools Used
- Google Earth Engine
- Remote Sensing (Sentinel-2)
- Machine Learning (Random Forest)
